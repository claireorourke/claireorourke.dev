# claireorourke.dev

Personal site for Claire O'Rourke. Built with React, TypeScript, and Vite. Deployed to GitHub Pages at [claireorourke.dev](https://claireorourke.dev).

## setup

```bash
npm install
```

## dev

```bash
npm run dev
```

## build and preview

```bash
npm run build
npm run preview
```

## deploy

```bash
npm run deploy
```
